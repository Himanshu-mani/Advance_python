# 🤖 Jarvis AI Core - Python OOP Journey

This repository tracks my progress in building a modular AI Assistant (Jarvis) while mastering Advanced Python and Data Structures.

## 📅 Day 1: Object-Oriented Programming (OOP) Fundamentals

### 🧠 Concepts Learned Today:
1.  **Classes & Objects**: Understanding the "Blueprint" (Class) vs the "Real Entity" (Object).
2.  **The `__init__` Method**: Using constructors to set up the initial state of an object.
3.  **The `self` Keyword**: Mastering the reference point that connects methods to their specific object.
4.  **CRUD Operations in OOP**:
    * **Create**: Adding tasks to a list within a class.
    * **Read**: Filtering and displaying tasks based on priority.
    * **Update**: Finding an item in a list and changing its attributes (e.g., priority).
    * **Delete**: Removing specific items from memory.
5.  **Data Management**: Using nested lists `[[task, priority]]` to store complex data.

### 🚀 Logic Milestone:
Built a `JarvisMemory` system that can store, filter, update, and delete tasks dynamically without messy global variables.
